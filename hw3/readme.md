# nn0 學習範例：基礎二分類模型

這個專案提供了一個使用自定義微型自動微分/神經網路庫 (`nn0`) 構建的基礎線性分類器範例。透過這個範例，你可以學習到機器學習中最核心的**前向傳播 (Forward Pass)**、**反向傳播 (Backward Pass)** 以及 **參數優化 (Optimization)** 流程。

## 📝 專案簡介

本範例訓練一個簡單的線性模型，用來解決一個二維空間中的二分類問題。
分類的數學規則為：當輸入特徵 $x_0 + x_1 > 0$ 時，歸類為 `類別 1`，否則歸類為 `類別 0`。

模型不依賴 PyTorch 或 TensorFlow，而是完全依賴 `nn0` 模組中的基礎節點 (`Value`) 來建立計算圖並自動計算梯度。

## 🧠 核心模組解析 (`nn0`)

程式碼依賴 `nn0` 提供的四個核心元件：
* **`Value`**: 自動微分節點。它不僅儲存純數值 (`.data`)，還能記錄該變數的梯度 (`.grad`) 以及追蹤運算歷史，以支援反向傳播。
* **`linear`**: 實現矩陣乘法。計算 $\text{logits} = W \cdot x$。
* **`cross_entropy`**: 交叉熵損失函數。評估模型預測的 Logits 分佈與真實標籤之間的差異，損失值越小代表預測越準確。
* **`Adam`**: 現代深度學習中最常用的優化器。根據計算出的梯度來更新權重參數。

## 🚀 程式碼流程步驟

### 1. 準備訓練數據
定義了 6 筆測試資料。每筆資料包含一個二維座標 `[x0, x1]` 與其對應的標籤 (`0` 或 `1`)。

### 2. 初始化模型參數
為了將二維輸入映射到兩個類別的得分 (Logits)，我們建立了一個 $2 \times 2$ 的權重矩陣 `W`。
這些權重都是被 `Value` 包裝的節點，初始值為 `-0.5` 到 `0.5` 之間的隨機數。

### 3. 訓練迴圈 (Training Loop)
模型共訓練 40 個 Steps (Epochs)。在每一個 Step 中，我們遍歷所有數據，並執行以下標準機器學習流程：
1. **前向傳播**: 將輸入 `x` 通過 `linear` 層，計算出對兩個類別的預測分數 `logits`。
2. **計算損失**: 使用 `cross_entropy` 計算預測結果與真實標籤 `target_idx` 之間的誤差。
3. **反向傳播**: 呼叫 `loss.backward()`。這會沿著計算圖往回推，計算出所有參與運算的權重（`W` 矩陣中的 `Value` 節點）的梯度。
4. **參數更新**: 呼叫 `optimizer.step()`。Adam 優化器會根據剛剛算出的梯度更新權重，並自動清空梯度以迎接下一次迭代。
   > **💡 技巧**: 程式碼中實作了「學習率衰減 (Learning Rate Decay)」，讓優化步長隨著時間逐漸變小，有助於模型在訓練後期穩定收斂。

### 4. 測試模型預測能力
訓練完成後，我們輸入兩個從未見過的新資料點：`[1.5, 1.0]` 與 `[-1.0, -1.0]`。
程式會比較計算出的兩個 Logits 分數，將分數較高的類別作為最終的預測結果。
